## 👾Getting Started

### Prerequisites

Whale Browser [설치](https://whale.naver.com/ko/download)가 필요합니다.

### Installing

```
npm install
```

## 🔌Running the tests

로컬에 프로젝트 저장 후, 웨일 브라우저 [개발자 모드](whale://extensions/)에서 프로젝트를 설치하면 테스트 가능합니다.
